## 过滤和验证

过滤和验证操作之间的差别，是过滤操作可能会除掉或转换某些值。而验证操作会使用与数据特点匹配的标准测试数据，并返回布尔型的结果。

1. 攻击防护和灵活性的要求-以一系列回调函数为基础的，更灵活的机制来执行过滤和验证的操作。
2. 为增加灵活性，应减轻过滤器和验证器基类的量级。
